# Amazon Clone

This is a simple Amazon Clone website created using HTML and CSS.
The project replicates the basic layout and design of the Amazon homepage to practice front-end development skills.

## Features

* Amazon-like homepage design
* Navigation bar with search option
* Product sections and banners
* Responsive layout

## Technologies Used

* HTML
* CSS

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run the **index.html** file in your browser.

## Project Purpose

This project was built to practice front-end web development and understand how e-commerce website layouts are created.
